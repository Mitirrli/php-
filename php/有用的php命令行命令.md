# 有用的php命令行命令

###### 1. 查看当前已安装的扩展
```
php -m 
```

###### 2. 查看当前php版本
```
php -v
```

###### 3. 开启服务 

```
在对应目录执行目录，例php -S 0.0.0.0:1234 即可在浏览器上访问了
```

###### 4. 快速查看扩展版本
  
```
php -i | less
输入 /扩展名 + 回车
```

###### 5. 查看php配置文件路径

```
php -i | grep php.ini
```

###### 6. 执行文件
```
应用:检查代码语法是否正确
php -f 文件名
```

###### 7. 直接执行代码
```
php -r 'echo 1;'
```

###### 8. 查看当前php可执行文件位置
```
which php
```
